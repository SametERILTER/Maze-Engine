# Maze Engine

Bu proje, temel labirent çözme algoritmalarının nasıl çalıştığını görsel olarak gösteren bir C++ uygulamasıdır.

## İçerik

Uygulama içerisinde aşağıdaki algoritmaların animasyonlu gösterimleri yer almaktadır:

- A* (A-Star)
- BFS (Genişlik Öncelikli Arama)
- DFS (Derinlik Öncelikli Arama)

## Kullanılan Araçlar

- C++
- SFML (Görsel kütüphane)

## Çalıştırma

Projeyi çalıştırmak için C++ derleyicisi ve SFML kütüphanesinin sisteminizde kurulu olması gerekmektedir. CMake kullanarak derleme yapabilir ve oluşan çalıştırılabilir dosyayı başlatabilirsiniz.

## Kullanım

Uygulama başladığında ekrandaki menüden bir algoritma seçerek labirentin nasıl çözüldüğünü izleyebilirsiniz.
